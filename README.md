<a name="quick-setup"></a>
## 🛫 Quick Setup

```sh
# clone the project
git clone https://github.com/ibrsec/redux-toolkit-mp3dowloader.git

# enter the project directory
cd redux-toolkit-mp3dowloader

# install dependency
npm install || yarn install

# develop
npm run dev || yarn start
```
